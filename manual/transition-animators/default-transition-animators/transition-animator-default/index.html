Default Transition Animator

To use a transition animator, you first create an instance of it, as described in Using A Transition Animator.

Description

The default transition animator presents content from the bottom of the screen, sliding it upwards. It dismisses content by performing the reverse - animating onscreen content downwards.

A default transition animator.


Configuration

The configurable properties of a default transition animator.



Name                     Description
Duration The duration of the transition.
Curve The animation curve of the transition.
Use Scaled Time Animate with scaled time. By default this transition uses unscaled time - i.e. it won't be affected by changes to 'Time.timeScale'. This allows it to animate when the game is paused, for example. Enable this if you want the transition to be affected by changes to 'Time.timeScale'.
